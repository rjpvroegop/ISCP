Gebruiken
De collectie in MongoDB veranderen 1. Klik op Settings
2. Klik op Change collection
Het zoekwoord veranderen 1. Klik op Settings
2. Klik op Change keyword
Een collectie verwijderen 1. Klik op settings
2. Klik op Remove collection
Huidige instellingen bekijken 1. Klik op Info
2. Klik op Check active settings
Bestaande collecties bekijken 1. Klik op Info
2. Klik op Check active collection
Hoe werkt de Plot De plot is opgebouwd als een soort heatmap
de rode stippen geven exact aan waar de tweet zich zou bevinden
het oranje vlak er omheen is als heatmap bedoeld voor visualisatie.
Hoe werkt de Graph De graph is opgebouwd uit meerdere regels
per regel maximaal 1.000 tweets. Bij de laatste regel staat hoeveel
tweets er in die regel zitten. Ook staat er een indicatie hoe
positief of negatief een tweet is.
